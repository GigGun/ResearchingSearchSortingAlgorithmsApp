# Программа для анализа эффективности алгоритмов поиска и сортировки массива целых чисел
## Описание проекта
Проект реализуется в рамках выполнения курсового проекта по предмету Объектно-ориентированне программирование</br>

Англ.название: ResearchingSearchSortingAlgorithmsApp </br>
Ссылка [Git](https://github.com/GigGun/ResearchingSearchSortingAlgorithmsApp). </br>

Данная программа предназначена для сравнительного анализа эффективности
различных алгоритмов поиска и сортировки путем сравнения времени работы,
необходимого для для выполнения алгоритма. </br>
Проект реализован на Python. Для реализации пользовательского интерфейса (UI)
использован фреймворк [PySide](https://www.qt.io/development/qt-framework/python-bindings). и включает реализацию нескольких
популярных алгоритмов с визуализацией процесса работы.
## Функциональность
### Алгоритмы поиска:
* Линейный (последовательный) поиск - O(n)
* Бинарный поиск - O(log n)
### Алгоритмы сортировки
* Сортировка вставками - O(n²)
* Быстрая сортировка - O(n log n) в среднем случае

## Алгоритмическая сложность
Для оценки верхней оценки границы (наихудшего случая) используется термин [**Big O**](https://habr.com/ru/articles/782608/).
### Поиск
| Алгоритмы | Лучший случай | Средний случай | Худший случай | Память |
| --------- |:-------------:|:--------------:|:-------------:|:------:|
| Линейный  | O(1)			| O(n)			 | O(n)			 | O(1)   |
| Бинарный  | O(1)			| O(n)			 | O(n)			 | O(1)   |
### Сортировка
| Алгоритмы | Лучший случай | Средний случай | Худший случай | Память     | Стабильность |
| --------- |:-------------:|:--------------:|:-------------:|:----------:|:------------:|
| Вставками | O(n)			| O(n²)			 | O(n²)		 | O(1)       | Да           |
| Быстрая   | O(n log n)    | O(n log n		 | O(n²)		 | O(n log n) | Нет          |
